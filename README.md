# Implement an identity management solution using Microsoft Entra ID

## Objective
Learn how to implement and configure Microsoft Entra ID for an organization. 
Set up an initial tenant, create and manage users and groups, configure external identities for collaboration, and implement hybrid identity solutions with Microsoft Entra Connect. 

### Skills Learned
- Implement identity and access management (IAM) in Microsoft Entra ID

### Tools Used
Microsoft Entra ID

## Products
Microsoft Entra External ID, Microsoft Entra Workload Identities
